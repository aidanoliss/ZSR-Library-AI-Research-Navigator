# Weekly Venture Radar Report Prompt

Generate a weekly report from discovered companies, scorecards, risk flags, domain summaries, investor matches, and proposal drafts.

Do not claim investment returns or provide investment advice. State limitations clearly.

## Required Markdown Structure

# Weekly Venture Radar Report

## Executive Overview

Brief explanation of what changed this week in the startup market.

## Best Startup by Domain

For each domain:

- Domain name
- Best startup
- Overall score
- Why it won
- Main risk
- Confidence level
- Suggested next step

## Top 10 Startups Overall

For each:

- Rank
- Company
- Domain
- Score
- One-sentence thesis
- Key evidence
- Main concern
- Recommended action

## Domain Momentum

For each domain:

- Number of new companies
- Average score
- Notable pattern
- Investor interest
- Market timing

## VC / Investor Outreach Targets

For each recommended investor:

- Investor name
- Type
- Relevant thesis
- Best-matched company
- Why they fit
- Suggested outreach angle

## Drafted Proposals

For each top company-investor match:

- Subject line
- Email body
- Short proposal memo
- Suggested ask

## Diligence Questions

Questions the user should investigate before making any recommendation.

## Limitations

State what the system could not verify and what requires human review.
